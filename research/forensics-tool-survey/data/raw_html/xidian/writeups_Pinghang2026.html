    Volatility3 的 filters 参数 (--filters)

    指定 filters 参数对 Volatility3 CLI 的输出进行过滤, 参数位置在插件名之前.

    语法: --filters [+-]column,pattern[!]

    • column,pattern: 过滤出列名 column 中包含 pattern 字符串的条目.
    • pattern!: 通过在 pattern 后添加 ! 以 pattern 进行正则匹配.
    • +: 包含匹配行, filters 默认行为
    • -: 排除匹配行
    • 若 column 缺省, 则会在所有列进行匹配.

    例如:

    • python vol.py -f mem.raw --filters "PID,1234" windows.pslist.PsList: 在 PsList 运行结果中过滤 PID(进程号) 为 1234 的条目.
    • python vol.py -f mem.raw --filters "Process,^Weixin\\.exe$!" windows.pslist.PsList: 在 PsList 运行结果中过滤 Process(进程名) 符合正则 ^Weixin\\.exe$ 的条目.
    • python vol.py -f mem.raw --filters "-PID,4" windows.pslist.PsList 在 PsList 运行结果中排除 PID(进程号) 为 4 的条目.
    • python vol.py -f mem.raw --filters "Process,^Weixin\\.exe$!" windows.pslist.PsList: 在 PsList 运行结果中过滤 Process(进程名) 符合正则 ^Weixin\\.exe$ 的条目.
    • python vol.py -f mem.raw --filters ",Weixin" windows.pslist.PsList: 在 PsList 运行结果中过滤包含 Weixin 的条目.
